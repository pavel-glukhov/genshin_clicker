## Telegram bot - автокликер для отметок Genshin Impact

### Описание

* Бот автоматически собирает награды раз в сутки по указанному времени. 

### Стек
1. Selenium
2. Aiogram
3. Apscheduler

### Запуск Бота:
 - добавить токен в .env по шаблону .env.example
 - Установить Docker
 - Выполнить - docker compose up
